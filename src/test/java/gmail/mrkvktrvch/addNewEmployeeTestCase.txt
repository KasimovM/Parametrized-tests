Заголовок: "Внесение информации о новом сотруднике в таблицу"
Приоритет: BLOCKER
Предусловия: Открыт сайт demoqa.com/webtables

Шаги:
1) Нажать кнопку [Add].
2) В поле "First Name" внести имя сотрудника.
3) В поле "Last Name" внести фамилию сотрудника.
4) В поле "Email" внести электроную почту сотрудника.
5) В поле "Age" внести возраст сотрудника.
6) В поле "Salary" внести зарплату сотрудника.
7) В поле "Department" внести название отдела, в котором работает сотрудник.
8) Нажать на кнопку [Submit].

Ожидаемый результат:
- В таблице появилась строка с информацией о новом сотруднике.
